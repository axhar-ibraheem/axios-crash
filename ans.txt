1. Why do we need headers any idea?

ans : Headers are used to send information that may be repeated across multiple requests or responses, their primary purpose is to provide context and information about the data being transmitted.

2. What is axios?
ans: Axios is a popular open-source JavaScript library that provides an easy-to-use interface for making HTTP requests from web browsers or Node.js. Axios supports a wide range of request methods, including GET, POST, PUT, DELETE, and more, and it provides features like automatic JSON parsing, request cancellation, and error handling.
